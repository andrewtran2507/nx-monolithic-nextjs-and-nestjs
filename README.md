```bash
NODEJS: v18.14.0
```

## API setup
```bash
1. npm install --global nx@latest
1. git clone <repo>
2. cd nx-monolithic-nextjs-and-nestjs
3. npm i
4. npm run fe:api:dev
5. HAPPY CODING
```
